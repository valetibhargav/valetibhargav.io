# Personal Portfolio Website

A responsive personal portfolio website for Bhargav Valeti, built with HTML, CSS, and JavaScript.

## Overview

This is a single-page personal portfolio designed to showcase who I am and the work I do. It features a responsive layout that adapts to different screen sizes, smooth navigation between sections, and a clean, modern design. The site is built entirely with front-end technologies and requires no build step.

## Sections

The site is organized into several sections: Home, About, Service, Experience, Portfolio, and Contact. It also includes links to social profiles such as LinkedIn, Twitter, Instagram, and Facebook.

## Built With

HTML, CSS, and JavaScript, along with Font Awesome for icons.

## How to Run

Clone or download this repository, then open `index.html` in any modern web browser. There is nothing to install.

## Project Structure

`index.html` contains the page markup, `style.css` handles the styling and responsive layout, `script.js` contains the interactivity, and the `images` folder holds the site's image assets.
